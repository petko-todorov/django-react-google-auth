# django-react-google-auth

Google OAuth Authentication System
A full-stack authentication system implementing Google OAuth 2.0 with Django REST Framework backend and React frontend. Supports both popup and redirect authentication flows.
Features

Google OAuth 2.0 Integration - Secure authentication via Google
Dual Authentication Flows - Support for both popup and redirect methods
Session Management - Server-side session handling with Django
Protected Routes - Authentication context for protecting React routes
CSRF Protection - Built-in CSRF token handling
User Session Verification - Automatic authentication state checking
